CONFEX: A Database for CONUS Fire EXtent
Raja Zubair Zahoor Qadiri, Diego Cerrai
Earth System Science Data Discussions
https://doi.org/10.5194/essd-2025-462
野火排放 卫星观测 火点探测
该论文（CONFEX: A Database for CONUS Fire EXtent）介绍了一个名为**CONFEX**的数据库，该数据库包含了美国本土和阿拉斯加地区2012年至2023年的**野火信息**。该数据库利用Suomi国家极轨合作卫星的**VIIRS S-NPP** 375米数据产品，通过DBSCAN聚类算法将热点位置聚类成实际的野火，并计算每个野火的范围、质心、起始和结束日期。通过与CALFIRE数据库的验证，证明该算法在空间和时间上准确聚合热点方面是有效的。该数据库对于研究野火趋势、风险、排放、空气质量以及规划具有重要意义，是一个公开可用的高分辨率< b >野火范围数据集。

空气质量监测 颗粒物 源解析
该研究论文（Implementation of real-time source apportionment approaches using the ACSM–Xact–Aethalometer (AXA) setup with SoFi RT: the Athens case study）主要关注城市空气污染，特别是**颗粒物**（PM）的来源解析。它介绍了一种实时监测系统，该系统使用先进的仪器（ACSM、Xact、Aethalometer）和软件（SoFi RT）来即时跟踪污染源。该系统通过化学、元素和黑碳数据的整合，能够对PM的来源进行连续、详细的分析。研究结果表明，交通排放是雅典PM的主要来源，其次是**二次物种**（如硫酸盐、硝酸盐、铵盐和二次有机气溶胶），生物质燃烧、烹饪、灰尘和海盐也有一定贡献。该研究强调了AXA系统在改善城市**空气质量监测**方面的潜力，并为有针对性地减少PM污染提供了信息。该研究通过实时监测和数据分析，为城市空气质量管理提供了有效的工具和方法。
